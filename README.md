# Simple Interest Calculator

A simple command-line application (or script) that calculates the simple interest based on the principal amount, annual interest rate, and time period in years.

## Formula Used
The simple interest is calculated using the standard formula:
A = P * (1 + R * T)

Where:
- **P**: Principal amount
- **R**: Annual interest rate (in decimal form, e.g., 5% = 0.05)
- **T**: Time period in years

## How to Use
1. Clone the repository.
2. Input the principal amount, rate of interest, and time duration when prompted.
3. The script will output the calculated interest and the final total amount.
